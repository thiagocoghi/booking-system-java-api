# 🏨 Sistema de Reservas

API REST em Java com Spring Boot para gerenciamento de reservas de recursos (quartos, mesas, assentos), com testes automatizados e documentação Swagger.

## 🚀 Tecnologias Usadas
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger
- JUnit 5
- RestAssured
- Selenium (opcional)

## 📦 Como Rodar o Projeto

```bash
./mvnw spring-boot:run
